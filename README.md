# RRRhody-API

## Running Locally

Make sure you have [Node.js](http://nodejs.org/) and the [Heroku Toolbelt](https://toolbelt.heroku.com/) installed.

```sh
$ npm i
$ npm start
```

## Development

```sh
$ npm i -g nodemon
$ npm run watch
```

The app will be running on `localhost:3838` by default

## Deploying to Heroku

```
$ git push heroku master
$ heroku open
```
